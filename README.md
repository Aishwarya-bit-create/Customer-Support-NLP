# Customer Support NLP Chatbot

## Project Overview
This project implements Natural Language Processing to automate customer support query classification and provide automated responses.

## Features
- Text classification using Machine Learning
- Streamlit web interface
- Automated response suggestion
- Customer query analytics

## Technologies Used
- Python
- NLP (NLTK)
- Scikit-Learn
- Streamlit
- Pandas

## How to Run

1. Clone repository
2. Install dependencies
   pip install -r requirements.txt

3. Run application
   streamlit run app.py

## Output
User can enter customer queries and get predicted category and response.
